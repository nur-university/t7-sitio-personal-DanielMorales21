Proyecto 1: Editor de multi-imagenes

Este proyecto consiste en una aplicación de escritorio desarrollada en Java que permite editar múltiples imágenes al mismo tiempo. El sistema fue diseñado para brindar una experiencia intuitiva y completa, permitiendo abrir, visualizar y modificar varias imágenes de forma independiente dentro del mismo entorno. Entre las herramientas integradas se incluyen funciones para aplicar filtros como oscurecer, aclarar, invertir colores, sepia, pixelar, convertir a escala de grises, binarización, y volteo tanto horizontal como vertical. También se incorporan herramientas de dibujo como puntos, líneas y rectángulos.

Una de las funcionalidades más destacadas del proyecto es el sistema de “deshacer” implementado de forma individual para cada imagen, utilizando estructuras de datos como pilas y colas para gestionar los cambios. Esto permite que el usuario revierta acciones específicas en una imagen sin afectar a las demás abiertas. Además, el sistema permite guardar las imágenes editadas, eliminar imágenes abiertas y visualizar el histograma RGB de cada una, ofreciendo un análisis visual de la distribución de los colores. El proyecto también incluye un registro (log) detallado de todas las acciones realizadas por el usuario para facilitar el seguimiento y control de los cambios. Esta aplicación representa un ejercicio completo de programación orientada a objetos y manejo de estructuras dinámicas en Java.

Proyecto 2: Arbol binario de busqueda

Este proyecto consiste en la implementación de un Árbol Binario de Búsqueda desarrollado en Java bajo el paradigma de programación orientada a objetos. El sistema tiene la capacidad de procesar archivos que contienen más de 67,000 palabras, cargándolas eficientemente dentro de la estructura del árbol. Una vez cargadas, permite al usuario visualizar las palabras en orden alfabético (recorrido inorden) o en un orden aleatorio, demostrando la flexibilidad del árbol.

El programa también ofrece funciones para agregar nuevas palabras al árbol y al archivo original, y una herramienta de búsqueda que permite localizar cualquier palabra. Esta búsqueda muestra en pantalla el número de saltos (comparaciones) necesarios para encontrar la palabra, ofreciendo una visión clara del rendimiento del algoritmo. Además, todas las operaciones realizadas quedan registradas en un log, lo cual permite llevar un historial de uso y facilita tareas de depuración o auditoría. Este proyecto fue fundamental para afianzar el uso de estructuras dinámicas, manejo de archivos y la eficiencia de algoritmos de búsqueda.

Proyecto 3: Agenda de contacto

Esta agenda de contactos es una aplicación de escritorio desarrollada en Java utilizando programación orientada a objetos. Permite al usuario crear, visualizar y gestionar una lista de contactos personales de forma sencilla y eficiente. Cada contacto contiene información esencial como nombre, correo electrónico y número de teléfono, los cuales pueden ser modificados o eliminados según necesidad.

Entre sus funcionalidades destacadas se encuentra la posibilidad de importar contactos desde archivos en formato VCF (vCard), lo que facilita la integración con otros sistemas y herramientas de contacto. También cuenta con la opción de abrir directamente WhatsApp Web desde la aplicación, agilizando la comunicación con los contactos almacenados. El sistema fue diseñado con una interfaz amigable y funcionalidades prácticas que simulan una agenda moderna, siendo un excelente ejercicio de manejo de estructuras de datos, lectura de archivos, eventos y ventanas en Java.
